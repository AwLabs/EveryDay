太多人有“想太多”的毛病。想做的事太多，下了班想健身锻炼身体、想写代码、想看书提高、想学英语，想做这想做那，结果什么都没做。  
三件小事，是最小行动。随手就能做，不需要思考，不给自己心理压力。20个俯卧撑，更进一步可以跟视频做锻炼；每天一题更进一步可以是5题10题。但我只要达到最小限度，今天的目标便达到了。
